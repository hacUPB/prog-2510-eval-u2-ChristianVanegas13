## 2. Las arquitecturas del computador.

### Arquitetctura cisc:
## Qué es CISC?

CISC es un sistema de instrucciones desarollados por intel que se requiere mucho tiempo para ser desarolladas. Al ser una coleccion de instrucciones con complejidades distinas, puede llegar a ser muy poco complejas a ser muy complejas.


Fuente: https://www.profesionalreview.com/2021/07/18/risc-vs-cisc/#Que_es_CISC


### Arquitectura risc
## Qué es RISC?

A diferencia de CISC, el RISC viene siendo una arquitectura de procesadores que se basa en colecciones un sistema mas simple que no llega a tener grado de dificultad complejas. Risc está hecho para minimizaer el tiempo limitando el numero de instrucciones.

el Risc debe de contener tres parametros que son buscar, deodificar y ejecutar. Tambien puede simplificar instrucciones complejas.

Fuente: https://www.profesionalreview.com/2021/07/18/risc-vs-cisc/#Que_es_CISC

## 3. ¿Qué es el hardware? 

la CPU tiene como componente principal, ejecutarinstrucciones para que el ordenar funcione sin problema.El CPU es el celebro maestro, la fuerza impulsora que trabaja incansablemente para que todo ordenador funcione correctamente. Tambien llamada processasor ó microprecesador, es el componente principal de cualquier ordenador, es como el corazón del computador que controla todo el sistema y asegurándose que todo funcione correctamente. La CPU es el que se encarga prácticamente de todas las tareas del procesador desde abrir git hasta solidworks.

### ¿Que es ALU?

Es una unidad aritmética lógica que hace estas operaciones con número enteros binarios, realmente hay procesos interesantes para poder hacer operaciones un poco más complejas con puntos flotantes. Pero bueno estas unidade utilizan números enteros binarios para hacer a grandes rasgos, esta es una unidad fundamental para que pueda existir el CPU.

### ¿Que es la Unidad de control?

La unidad de control es la que dirige las operaciones más importante del ordenador, estableciendo la comunicación entre ALU, la memoria principal y el resto de componentes. Controla la ejecución de cada dirección de un programa.

### ¿Que son los Registros?

Es una memoria que guarda operaciones de manera temporal y se dividen en registros de estado, este registros tiene un control sobre las unidades de registros. 

- Los registros de usos general se dividen en 4:AX,BX,CX,DX. AX es llamado acumulador, BX llamado índice base, CX llamado controlador y DX llamado datos.

- Los registros de segmento CS llamado código, DS llamado datos, SS llamado pila y ES llamado segmento extra.

- Registros de puntero es un registro de puntador y de índice cada una se usa para distinta direcciones IP apuntador de instrucciones, SP apuntador de pila, BP apuntador de base, SI apuntador de fuente y DI apuntador de destino.

### ¿Que son los Buses?

- Transportan datos y señales de control entre la CPU y otros componentes.

## ¿Que es GPU?

GPU (Unidad de Procesamiento Gráfico). La GPU es un componente que se encarga de procesar y renderizar gráficos e imágenes. Es ideal para tareas que requieren procesamiento paralelo, como videojuegos y aplicaciones de inteligencia artificial.

- Funcionamiento: Las GPU tienen muchos núcleos de procesamiento que trabajan en paralelo para manejar grandes volúmenes de datos al mismo tiempo.

#### La comparación GPU vs CPU:

- Función Principal: La GPU procesa gráficos; la CPU ejecuta y coordina el sistema.

- La GPU está optimizada para tareas simples en paralelo; la CPU para tareas complejas secuenciales.

- Núcleos de Procesamiento: La GPU tiene cientos o miles de núcleos; la CPU tiene pocos núcleos (2-16).

#### Diferencias:

La GPU tiene muchos núcleos, la CPU tiene menos, pero más rápidos.

La GPU se usa para gráficos intensivos, la CPU para aplicaciones generales.

EN rendimiento la GPU es mejor para tareas paralelas, la CPU para tareas secuenciales.

## ¿Cuales son las memorias?

##### Registros

Almacenan datos temporales dentro de la CPU y son muy rápidos.

##### Caché

Memoria rápida que guarda datos de uso frecuente para que la CPU los acceda rápidamente.

##### Principal (RAM)

Almacena datos y programas que se están usando en el momento; pierde los datos cuando se apaga la computadora.

##### Secundaria (Disco Duro y Unidades Externas)

Almacenan datos de forma permanente, tienen gran capacidad pero son más lentos que la RAM.

### (e)

## Buses de Datos:

Son sistemas que permiten la transferencia de datos entre los componentes de una computadora.

- Bus de Datos: Transporta los datos entre la CPU, la memoria y otros componentes.

- Bus de Direcciones: Transporta las direcciones de memoria donde los datos deben ser leídos o escritos.

- Bus de Control: Transporta señales de control y sincronización entre los componentes.


## 4. ¿Que es el software?

Es un conjunto de aplicaciones y programas diseñados para cumplir diversas funciones dentro de un sistema. Son los programas y los sistemas operativos que permiten que la computadora pueda diseñar tareas inteligentes, dirigiendo a los componentes físicos con 

instrucciones y datos a través de diferentes tipos de programas.

### (a) Software del sistema:

El software del sistema es un conjunto generalizado de programas, que gestionan los recursos de una computadora.como el Procesador central, los enlace de comunicación y dispositivo perisfericos.

Tiene varios sistemas como por ejemplo **el sistema operativo** que gestiona y controla las actividades de la computadora, programa traductores que es prácticamente lo que convierte el lenguaje de programación a lengua de máquina.

### (b) Software de aplicaciones:

El software de aplicaciones es el conjunto de programas escrito por empresas o usuarios individuales ó en equipo que instruyen a la computadora para ejecutar una tarea específica.

**Ejemplos**:_Word, Word perfect, excel, power point.

## (c) Software de desarrollo:

 Es prácticamente empresas o usuario individuales que crean empresas desarrollando procesadores, con la cual crean programas como _Java_ y más para hacer más fácil el uso de los programas de computadores.

## 5. Funcionamiento del Computador

### ¿Qué procesos se llevan a cabo cuando se enciende una computadora?

Cuando enciendes una computadora, pasan varias cosas:

- 1.Se enciende la fuente de alimentación.

- 2. La BIOS hace un chequeo rápido del hardware (POST).

- 3. La BIOS o UEFI inicializa los dispositivos.

- 4. Carga el sistema operativo desde el almacenamiento a la RAM.

- 5. El sistema operativo se inicia y carga los controladores y servicios.

### ¿Qué sucede desde que ingreso un dato a través del teclado, hasta que veo el resultado de la operación en la pantalla?

- 1. Pulsas una tecla.

- 2. El teclado envía una señal a la CPU.

- 3. La CPU procesa la señal y convierte el dato.

- 4. El dato se almacena temporalmente en la RAM.

- 5. La CPU ejecuta las instrucciones correspondientes.

- 6. La GPU muestra el resultado en la pantalla.

### ¿Cómo se codifican los datos internamente en el computador?

Los datos se codifican en binario (0 y 1). Los bits se agrupan en bytes (8 bits), y estos bytes representan texto, imágenes, sonidos, etc.

### ¿Cuáles son las unidades de medida de datos en un computador?

- Bit: Unidad más pequeña (0 o 1).

- Byte: 8 bits.

- Kilobyte (KB): 1,024 bytes.

- Megabyte (MB): 1,024 KB.

- Gigabyte (GB): 1,024 MB.

- Terabyte (TB): 1,024 GB.
